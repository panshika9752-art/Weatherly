# 🌦️ Rapid Forecast

A modern and responsive weather application that provides real-time weather updates for any city around the world.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature updates
- 💧 Humidity information
- ☁️ Weather condition display
- 📱 Responsive design for all devices
- ⚡ Fast and user-friendly interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- OpenWeather API
- Git & GitHub
- Vercel
